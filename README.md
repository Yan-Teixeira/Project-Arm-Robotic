# Project-Arm-Robotic

O objetivo desse projeto é desenvolcer um braço robótico de automação totalmente feito em impressão 3D, focado para agro. Nesse repositória estará todas atualizações do projeto, com a documentação e códigos. A modelagem 3D, a documentação, os códigos e os matariais escolhidos foram utilizados do repositório oficial do desenvolvedores do projeto Thor.  

Link do projeto Thor: https://github.com/AngelLM/Thor
